# Glossaire — Technologies clés derrière les chatbots modernes

| **Technologie**                                      | **Rôle**                                                                 |
| ---------------------------------------------------- | ------------------------------------------------------------------------ |
| **NLP** (Natural Language Processing)                | Comprendre et traiter le langage humain                                  |
| **NLU** (Natural Language Understanding)             | Comprendre l'intention et le contexte                                    |
| **NLG** (Natural Language Generation)                | Générer des réponses en langage naturel                                  |
| **Machine Learning**                                 | Apprendre à partir des données sans programmation explicite              |
| **Deep Learning**                                    | Réseaux neuronaux profonds pour apprentissage complexe                   |
| **Transformers**                                     | Architecture de base des LLM modernes (GPT, BERT, Claude, Gemini, etc.) |
| **LLM** (Large Language Models)                      | Modèles entraînés sur d'énormes quantités de texte                       |
| **Reinforcement Learning**                           | Apprentissage par récompense pour améliorer les réponses                 |
| **Tool Use / Function Calling**                      | Capacité d'utiliser des outils et APIs externes                          |
| **Memory Systems**                                   | Mémoire à court et long terme pour contexte persistant                   |
